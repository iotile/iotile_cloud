You will need internet access to complete the device set up and activation. If your device monitoring location has no internet connectivity, claim the device on a location with internet location before installing the device on its final location.

  • 1

    Open the installed mobile app and login using your IOTile Cloud credentials.


  • 2

    Using the side menu, select 'New Device Setup'


  • 3

    Select your device from the scanned devices by matching the ID number



    Note: If you don't see your device, make sure the battery is installed and you are within 50 feet.

  • 4

    Follow the 'New Device Setup' to create a project and configure the settings for both your project and device.


    That's it!

    For more information about all the things you can do with your IOTile Device, see our user guide page and refer to documentation that came with your device.
